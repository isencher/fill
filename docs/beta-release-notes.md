# Fill v0.1.0-beta 发布说明

## 🎉 欢迎！

欢迎使用 **Fill v0.1.0-beta** - 智能表格数据填充工具

Fill 可以帮助您：
- 📤 **上传数据**：支持 CSV 和 Excel 文件
- 🎯 **智能映射**：自动识别并映射字段名
- 📋 **使用模板**：内置发票、合同、报表等模板
- ⚡ **批量处理**：一次性生成多个文件
- 📥 **下载结果**：批量下载所有输出

---

## 🚀 快速开始

### 第 1 步：准备数据
- 准备包含数据的 Excel 或 CSV 文件
- 确保文件有清晰的列名（第一行）

### 第 2 步：上传文件
- 拖拽文件到上传区域
- 或点击"选择文件"按钮
- 等待上传完成

### 第 3 步：选择模板
- 从内置模板中选择一个
- 或上传您自己的模板文件

### 第 4 步：确认映射
- 检查自动映射结果
- 手动调整映射关系
- 点击"确认"生成预览

### 第 5 步：生成下载
- 确认预览结果
- 点击"生成"处理文件
- 等待完成后下载

## ⚠️ Beta 版本说明

### 已知限制

1. **文件大小限制**：最大 10MB
2. **行数限制**：建议 10,000 行以内
3. **支持格式**：CSV, XLSX, XLS

### 浏览器兼容性

✅ **支持**：
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

⚠️ **部分支持**：
- 移动浏览器（功能完整但体验一般）

❌ **不支持**：
- IE 11

## 🐛 遇到问题？

### 常见问题

**Q: 上传失败？**
```
A: 请检查：
  • 文件格式是否支持（CSV/XLSX/XLS）
  • 文件大小是否超过 10MB
  • 网络连接是否正常
```

**Q: 映射不准确？**
```
A: 可以：
  • 手动调整映射关系
  • 点击列标题查看详情
  • 使用拖拽重新排序
```

**Q: 生成失败？**
```
A: 请确认：
  • 模板文件格式正确
  • 映射关系完整
  • 数据列数匹配
```
